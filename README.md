# sistema-consultorio-cpp

Sistema integral desarrollado en **C++** para la administración de turnos, pacientes y profesionales médicos.

##  Desafío Técnico
El objetivo principal fue implementar un sistema de almacenamiento propio mediante el manejo de **archivos binarios**, profundizando en la gestión de estructuras de datos y persistencia en disco sin depender de motores de bases de datos externos.

##  Características
* **Gestión CRUD:** Control total de Pacientes, Médicos y Turnos.
* **Persistencia:** Lectura y escritura de registros en archivos `.dat`.
* **Lógica:** Algoritmos de búsqueda, filtrado y ordenamiento.
* **Validación:** Sistema robusto de entrada de datos por consola.

##  Tecnologías
* C++
* IDE: Code::Blocks
* Manejo de Archivos (I/O)
